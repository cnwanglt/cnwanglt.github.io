申请SAC

Seismic Analysis Code 是一款开源的，配置在Linux上面进行使用的软件包，同时也是天然地震学领域使用最广泛的数据分析软件包之一。

第一步我们需要在iris社区申请软件包(这里是软件包申请地址)

认真填写个人信息，尤其注意Email那一栏，需要填写单位邮箱。如果使用QQ、163这样的邮箱很容易被直接拒绝。如果没有单位邮箱，需要提供其他信息来辅助验证你的身份。

我在网站申请的是Linux_64位版本。
如果你在申请时选择了多个版本，对方的工作人员会想你询问为何你会同时用到这些（别问我怎么知道的）。

申请提交之后，需等待两三个工作日的审核，没什么问题的话就可以通过邮箱收到软件包了。
