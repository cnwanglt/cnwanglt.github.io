我的第一篇博客

发表于 2018-01-29 | 分类于 杂谈

Hello Github World！


Hello Github World！！
长期以来，编程界都认为刚接触一门语言时，如果首先使用它来编写一个在屏幕上显示消息“Hello World” 的程序，将会带来好运。

我是谁？

好了，先介绍下我自己，我叫汪龙潭，即将开始我的北京大学地球物理学研究生的学习阶段。编程语言只会用c/c++和Python，当然，只是会用而已，谈不上精通。虽然生在地球物理学专业，但我也有一颗成为一名优秀程序员的心。
2017年刚刚过去。对我来讲是不平凡的一年。大学生活渐入尾声，最开始忙着联系推免学校，联系导师，而后体验生活似的参与了些校招及一些程序员岗位招聘，仔细品读了大神们的博客后，我喜欢上了这种表达与分享的方式。在我看来，博客的记录方式要对的住写的本身，因而更多的去记录一些值得值得分享，有思考意义的内容。

所以，我写博客的动机主要是：

*·积累* 积累和转化自己所掌握的前沿技术
*·表达* 练习我的表达能力，把事情讲清楚
*·展示* 作为展示我的个人素质的平台

我的博客内容

就像刚才说的，为了对得起写的价值，我会更多的记录和分享我所掌握的前沿技术，做产品过程中得到的感悟，以及对人生的思考，并从中锻炼自己的表达能力。我文笔不佳，因而不会走一些文艺路线，也不会碎碎念的写一些生活琐事。此外，我还会转载一些我认为精彩的博客，来丰富内容和提高我的博客的水平，也能更好的促进交流。
